1. 物件除了擁有屬性以外，還可以擁有一些方法(method)，這些方法都是表現物件本身會有的行為、功能，每個方法都是由特定的函式所構成。





2. 定義一個物件所擁有的方法，可以像是：
```
let/const objName = {
	property1: value1,
	property2: value2,
		.
	propertyN: valueN
	
	method1: function funName(variable1, variable2, ..., variableN) {
					
					statemenet
					return value/expression
		 }
	method2: function (variable1, variable2, ...., variable N) {
					statement
					return value/expression
		 }
}
```
